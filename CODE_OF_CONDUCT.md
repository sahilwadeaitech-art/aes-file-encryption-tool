# Code of Conduct

## Summary

Be respectful. Be constructive. Don't be a jerk.

## Details

This project is open to anyone who wants to contribute or learn from it. I expect:

- Respectful communication in issues and PRs
- Constructive feedback (not personal attacks)
- No spam, harassment, or trolling
- Credit where credit is due

## Scope

This applies to GitHub issues, pull requests, discussions, and any other project-related communication.

## Enforcement

If someone's being disruptive, I'll address it directly. Repeated violations = blocked.

## Questions

Open an issue or reach out directly.
